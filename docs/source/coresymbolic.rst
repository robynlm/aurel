symbolic_descriptions

.. _symbolic_assumed_quantities:

Assumed quantities
==================

If not defined, vacuum Minkowski is assumed for the definition of the following quantities:

.. _aurel.core.AurelCoreSymbolic.gdown:

`gdown <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.gdown>`_: Metric tensor in the down index form.  I assume Minkowski, if not then please define AurelCoreSymbolic.data['gdown'] = ...

Callable quantities
===================

.. _aurel.core.AurelCoreSymbolic.gup:

`gup <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.gup>`_: Metric tensor in the up index form

.. _aurel.core.AurelCoreSymbolic.gdet:

`gdet <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.gdet>`_: Determinant of the metric tensor

.. _aurel.core.AurelCoreSymbolic.Gamma_udd:

`Gamma_udd <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Gamma_udd>`_: Christoffel symbols in the up-down-down index form

.. _aurel.core.AurelCoreSymbolic.Gamma_down:

`Gamma_down <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Gamma_down>`_: Christoffel symbols in the down index form

.. _aurel.core.AurelCoreSymbolic.Riemann_down:

`Riemann_down <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Riemann_down>`_: Riemann curvature tensor in the down index form

.. _aurel.core.AurelCoreSymbolic.Riemann_uddd:

`Riemann_uddd <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Riemann_uddd>`_: Riemann curvature tensor in the up-down-down-down index form

.. _aurel.core.AurelCoreSymbolic.Ricci_down:

`Ricci_down <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Ricci_down>`_: Ricci curvature tensor in the down index form

.. _aurel.core.AurelCoreSymbolic.RicciS:

`RicciS <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.RicciS>`_: Ricci scalar

.. _aurel.core.AurelCoreSymbolic.Einstein_down:

`Einstein_down <../_modules/aurel/coresymbolic.html#AurelCoreSymbolic.Einstein_down>`_: Einstein tensor in the down index form
